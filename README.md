# Trilha JS Developer - Pokedex

# Desafio Pokedex da Dio
O objetivo desse desafio é criar uma pagina de detalhes sobre o pokemon da primeira geração ao clicar sobre o pokemon que esta na lista.

## Instruções 
1. Clone o projeto localmente na sua maquina utilizando o comando: `git clone https://github.com/felipertec/js-developer-pokedex.git` ou você pode fazer o download do arquivo zipado clicando em download zip

2. Com o projeto em sua maquina execute o arquivo index.html executar

## Aprendizado
Nesse projeto foi abordado os conceitos de javascript e interação com a API pokeapi.

os conceitos abordados foram:
1. Variaveis
2. Condicionais
3. Promises
4. Fetch
